# 4. OBSERVATION WING

## Layout

Containment rooms on both sides.

Ground hallway between them.

Above:

Observation catwalk.

Both run same direction.

Players choose.

---

# OPTION A — Hallway Route

Pros:

- Faster
- More cover
- Easier movement

Cons:

* Lights in observation room come on as you get to the window of each room
- Doors to containment rooms can malfunction and open

Threat:

Centaurs can burst out.

---

# OPTION B — Catwalk Route

Pros:

- Can see into rooms below
- Can spot danger before it opens
- Easier ranged advantage

Cons:

- Unstable
- Risk of falling

Threat:

Collapse into rooms.

---

# How to make it feel balanced

Important:

Before they choose:

They can SEE:

Hallway:

- Stable ground
- But damaged door locks sparking

Catwalk:

- High visibility
- But rusted supports

Both are clearly risky.

Equal choice.

Perfect.
